# RAG Question Answering Bot

This project implements a Retrieval-Augmented Generation (RAG) pipeline that answers questions from documents using embeddings and vector search.

## Features
- PDF document loading
- Text chunking
- Embeddings
- FAISS vector database
- LLM-based answer generation

## Tech Stack
- Python
- LangChain
- FAISS
- OpenAI API

## Run locally

pip install -r requirements.txt  
python main.py

A sample document (sample.pdf) is included for testing.
